# 結論

個人間送金は **ことら送金** がベスト！！

### メリット
- **手数料無料**で銀行口座に送金できる！
    - 銀行にお金が入るのが一番嬉しいですよね？！
        - PayPayに何万円も残高があっても困る
        - とはいえ現金は小銭とか煩雑
    - 通常、銀行振込はお金がかかる
        - 「ことら送金」では送金手数料が原則無料

### デメリット
- 利用できる金融機関が限られる
    - 自分の口座、相手の口座がともに対応している必要がある
    - しかし！！多くの銀行は対応している
        - 三菱UFJ銀行, ゆうちょ銀行, 三井住友銀行, みずほ銀行, etc.
        - ※ 各金融機関、対応するアプリが必要
- 金額の制限がある
    - 1日あたり10~50万円(金融機関により異なる)の上限がある
    - でも、別に十分じゃない?!
