# 結論

個人間送金は **ことら送金** がベスト！！
https://www.cotra.ne.jp/p2pservice/

### メリット
- **手数料無料**で銀行口座に送金できる！
    - 銀行にお金が入るのが一番嬉しいですよね？！
        - PayPayに何万円も残高があっても困る
        - とはいえ現金は小銭とか煩雑
    - 通常、銀行振込はお金がかかる
        - 「ことら送金」では送金手数料が原則無料

### デメリット
- 利用できる金融機関が限られる
    - 自分の口座、相手の口座がともに対応している必要がある
    - しかし！！多くの銀行は対応している
        - 三菱UFJ銀行, ゆうちょ銀行, 三井住友銀行, みずほ銀行, etc.
        - ※ 各金融機関、対応するアプリが必要
- 金額の制限がある
    - 1日あたり10~50万円(金融機関により異なる)の上限がある
    - でも、別に十分じゃない?!

# 

飲食店などで代表してお金を払ったとき、どのようにお金を回収していますか？  
基本的にはPayPayで送金する人が多いでしょう。 
**しかし!!!** PayPayの残高がいっぱいあっても困りませんか？  
だからといって現金で集めるのは小銭の処理とか色々めんどくさい...
本当は現金で燃え合うのが一番嬉しいけど、それはめんどくさい。さてどうするべきか...  

といったあるあるを解決できるのが **「ことら送金」** です。  
手数料無料で、ほとんどの金融機関が対応しており、意識してPayPayを使う、いっそのこと手数料を払って銀行に出金するといったストレスから解放されます。

## なぜ「ことら送金」が最強なのか？

### 1. 「電話番号」や「メールアドレス」で送れる

銀行振込で一番面倒なのは、**「店番号・口座番号・名義」を聞き出すこと**ではないでしょうか？
ことら送金なら、相手が事前に設定していれば、電話番号だけで送金が完了します。

> **Tips:** 振込先を教える側も、口座番号というデリケートな情報を伝えなくて済むので心理的ハードルが下がります。

### 2. リアルタイム着金

通常の銀行振込（モアタイムシステム等）も早くなりましたが、ことら送金も基本24時間365日、即座に相手の口座に反映されます。飲み会の翌朝にサクッと送って、相手もその場で確認できるスピード感は、まさにキャッシュレス時代の割り勘に最適です。

## 実践！スマートな回収フロー

私が実践している、一番スムーズな回収手順を紹介します。

1. **代表者が一括で支払う**
* ポイントも貯まるし、レジが混まないのでスマート。


2. **グループチャットに「支払い依頼」を投げる**
* 「一人 3,500円です！ことら送金かPayPayでお願いします」と送る。


3. **「ことら」を優先してもらう**
* 「銀行口座に直接入るから、できれば『ことら』が助かる！」と一言添えるのがコツ。


## 注意点：アプリの確認が必要

「ことら送金」という名前のアプリがあるわけではありません。
銀行のアプリの中に、ことら送金の機能が組み込まれています。

* 三菱UFJ銀行なら「BankPay」や「三菱UFJ銀行アプリ」
* 三井住友銀行なら「BankPay」や「三井住友銀行アプリ」
* ゆうちょ銀行なら「ゆうちょ通帳アプリ」


# まとめ

PayPayは手軽さで勝りますが、間違いなく銀行口座に直結する **ことら送金** が流動性の面で勝っていますね！

特に、数千円〜数万円単位の大きな金額が動く旅行や飲み会では、この「手数料無料の口座間移動」が大きな威力を発揮します。まだ使っていない友人がいたら、ぜひこの記事をシェアして教えてあげてください！
